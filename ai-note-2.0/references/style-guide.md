# AI Note 风格指南

## 使用场景与风格

| 内容类型 | 推荐风格 | 文字编辑手法 | 图解脱稿手法 |
|---------|---------|------------|------------|
| 技术原理 | 蓝紫渐变科技风 | 概念卡片、流程图、对比表 | Mermaid流程图、架构图 |
| 学习笔记 | 暖色手写风 | 高亮标注、层级列表 | 思维导图、时间线 |
| 商业分析 | 简约商务风 | 数据表格、要点列表 | 柱状图、饼图、流程图 |
| 知识科普 | 清新自然风 | 问答式、逐步拆解 | 序列图、思维导图 |
| 产品介绍 | 品牌色卡片风 | 特性卡片、对比表格 | 架构图、功能脑图 |

## 文字笔记结构模板

```
## [分类] 核心主题标题

### 1. 概念一
一句话定义。
- 关键点1
- 关键点2

### 2. 概念二
...

### 3. 总结要点
```

## 图解笔记结构模板

### 模块1: 整体流程图
```
flowchart TB
  A[起点] --> B[步骤1]
  B --> C[步骤2]
  C --> D[终点]
```

### 模块2: 思维导图
```
mindmap
  root((主题))
    分支1
      子项1
      子项2
    分支2
```

### 模块3: 关系/对比图
使用CSS卡片布局展示对比关系

## 配色参考

- **主色**: #667eea (蓝色), #764ba2 (紫色)
- **强调色**: #22c55e (绿色), #f59e0b (橙色), #ec4899 (粉色)
- **背景色**: #f0f2f5 (浅灰), #fff (白色)
- **文字色**: #1a2332 (深色), #3a4a62 (正文), #5a6a7e (辅助)
